Learn JavaScript & jQuery with Flashcards!

JavaScript

JavaScript is a programming language used to make web pages interactive.

jQuery

jQuery is a JavaScript library that makes it easier to incorporate JavaScript on your web pages. The goal of jQuery is to allow its users to "write less, do more".

Function

A function is a block of code that performs an action and returns a result; optionally takes arguments

Method

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Argument

Arguments provide a bit more information to methods to help them know what they're supposed to do. Some methods take one or more arguments.

Parameter

Parameters are used to define a function, whereas arguments are used to invoke a function.

Return Value

The return value is the method's response.

Chaining Methods

Chaining methods can be used to simplify code and is a technique that allows you to call multiple functions on the same object.

String

The string type represents text.

Boolean

Booleans can only hold 2 possible values - true or false.

Undefined

The undefined data type has only one possible value - undefined. When declaring a variable without giving it a value (var myNumber;), JavaScript creates the variable without a value, hence undefined.

NaN

Stands for "not a number", but is considered a number.

Operator

There are several types of operators. A couple of common types are the assignment and comparison operators. An assignment operator changes the value of the variable on the left of the operator. A comparison operator does not change any values, but returns a boolean (true or false) depending on whether the statement evaluates as true or false.

Alert

An alert is used when you want to make sure information comes through to the user. It has a specified message that will only go away when the box is closed.

Comments

Anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

Variable

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name. Also, always use the var keyword when initializing variables in JavaScript.
